## Docker Compose command

### run command

```
docker compose up
```

### remove command

```
docker compose down
```

### check on images

```
docker ps
```

### Network config check

```
docker network <netwrok-name>
```

### sh into partciular image

```
docker exec -it <image Id> sh
```

### ping command

```
ping ipaddress <image-name>
```
